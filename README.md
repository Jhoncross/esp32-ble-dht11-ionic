# ESP32 BLE DHT11 Ionic

Proyecto que implementa comunicación Bluetooth Low Energy (BLE) entre un ESP32 y una aplicación móvil desarrollada en Ionic.

## 🔧 Tecnologías usadas
- ESP32
- DHT11
- Ionic
- Capacitor
- Bluetooth Low Energy (BLE)

## 📱 Funcionalidades
- Escaneo de dispositivos BLE
- Conexión al ESP32
- Recepción de temperatura y humedad
- Visualización en tiempo real
- Almacenamiento de datos
- Gráficas estadísticas
